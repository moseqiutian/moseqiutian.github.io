一、Date（日期）对象


1、Date 对象的声明


Date 对象用于处理日期和时间。


2、创建 Date 对象的语法：


var myDate=new Date()


3、Date 对象方法


a.Date()：方法可返回当天的日期和时间。 


实例
			document.write(Date())
			
结果

b.getDate() 方法可返回月份的某一天。 


实例:我们将输出当前月份的日期：

             var d = new Date()
             document.write(d.getDate())
			
结果

c.getDay() 方法可返回表示星期的某一天的数字。返回值是 0（周日） 到 6（周六） 之间的一个整数。 


实例:我们将以数字取得星期的当前一天

			var d = new Date()
			document.write(d.getDay())
			
结果

d.getHours() 方法可返回时间的小时字段。返回值是 0 （午夜） 到 23 （晚上 11 点）之间的一个整数。 


实例:我们可取得当前时间的小时：

			var d = new Date()
			document.write(d.getHours())
			
结果

e.getMinutes() 方法可返回时间的分钟字段。返回值是 0 ~ 59 之间的一个整数。 


实例:我们可取得当前时间的分钟：

			var d = new Date()
			document.write(d.getMinutes())
			
结果

f.getMonth() 方法可返回表示月份的数字。返回值是 0（一月） 到 11（十二月） 之间的一个整数。 


实例:我们将取得当前的日期，并把它输出：

			var d = new Date()
			document.write(d.getMonth()+1)
			
结果

g.getSeconds() 方法可返回时间的秒。返回值是 0 ~ 59 之间的一个整数。 


实例:我们可取得当前时间的秒：

			var d = new Date()
			document.write(d.getSeconds())
			
结果

h.getTime() 方法可返回距 1970 年 1 月 1 日之间的毫秒数。指定的日期和时间距 1970 年 1 月 1 日午夜（GMT 时间）之间的毫秒数。 


实例:我们将取得从 1970/01/01 至今的毫秒数，并输出它：

			var d = new Date()
			document.write(d.getTime() + " milliseconds since 1970/01/01")
			
结果

i.getFullYear() 方法可返回一个表示年份的 4 位数字。返回值是一个四位数，表示包括世纪值在内的完整年份，而不是两位数的缩写形式。 


实例:我们将取得当前的年份，并输出它：

			var d = new Date()
			document.write(d.getFullYear())
			
结果

二、Math 对象


1、Math 对象声明


Math 对象用于执行数学任务。


2、创建Math 对象的语法：


var pi_value=Math.PI;


3、Date 对象方法


a.random() 方法可返回介于 0 ~ 1 之间的一个随机数。返回值为0.0 ~ 1.0 之间的一个伪随机数。


实例:我们将取得介于 0 到 1 之间的一个随机数：

			document.write(Math.random())
			
结果

b.JS的随机数函数及相关函数：


1.Math.random(); 结果为0-1间的一个随机数(包括0,不包括1) 。


2.Math.floor(num); 参数num为一个数值，函数结果为num的整数部分。


3.Math.ceil(n); 返回大于等于n的最小整数。


4.Math.round(num); 参数num为一个数值，函数结果为num四舍五入后的整数。


c.可以用以上函数配合实现取x到y的随机数：


1.用Math.ceil(Math.random()*y);时，主要获取1到y的随机整数，取0的几率极小。


2.用Math.round(Math.random()*(y-1) + x)，可基本均衡获取x到y的随机整数，其中获取最小值x和最大值y的几率少一半。


3.用Math.floor(Math.random()*y + x);时，可均衡获取x到y的随机整数。
