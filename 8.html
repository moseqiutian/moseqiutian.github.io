一、drawImage() 方法


1.1、定义跟用法


drawImage() 方法在画布上绘制图像、画布或视频。


drawImage() 方法也能够绘制图像的某些部分，以及/或者增加或减少图像的尺寸。


1.2、JavaScript 语法 1


在画布上定位图像：


context.drawImage(img,x,y);


1.3、JavaScript 语法 2


在画布上定位图像，并规定图像的宽度和高度：


context.drawImage(img,x,y,width,height);


1.4、JavaScript 语法 3


剪切图像，并在画布上定位被剪切的部分：


context.drawImage(img,sx,sy,swidth,sheight,x,y,width,height);

        var left = document.getElementById('left');
        left.onclick=function () {
          var x = c.width/2; //画布宽度的一半
          var y = c.height/2;//画布高度的一半
           ctx.clearRect(0,0, c.width, c.height);  //清除画布内的东西
           ctx.translate(x,y);     //获得圆心点为中心点
           ctx.rotate(-90*Math.PI/180);   //旋转90°
           ctx.translate(-x,-y);       //更改圆心点回0,0
           ctx.drawImage(img,0,0,200,200);   //绘画出图片
        }

        var right = document.getElementById('right');
        right.onclick=function () {
          var x = c.width/2; //画布宽度的一半
          var y = c.height/2;//画布高度的一半
           ctx.clearRect(0,0, c.width, c.height);
           ctx.translate(x,y);
           ctx.rotate(90*Math.PI/180);
           ctx.translate(-x,-y);
           ctx.drawImage(img,0,0,200,200);
        }
